## Project setup
*you will need a .env file in your root directory with the necessary environment variables*
```
yarn setup
```

### Compiles and hot-reloads for development
```
yarn electron:serve
```

### Run your unit tests
```
yarn test:unit
```

### Run your end-to-end tests
```
yarn test:e2e
```
